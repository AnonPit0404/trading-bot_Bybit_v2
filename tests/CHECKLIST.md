# Чек-лист тестирования стратегии Spot DCA v2

## 1. Интеграция и запуск (Docker)
- [ ] Контейнер запускается без ошибок (exit code 0).
- [ ] Бот успешно подключается к API Bybit (проверка баланса).
- [ ] Логи выводятся в консоль в реальном времени.

## 2. Торговая логика (EMA + RSI)
- [ ] Сигнал на вход срабатывает только при пересечении EMA (fast > slow).
- [ ] Фильтр RSI (45-72) отсекает входы в зонах перекупленности/перепроданности.
- [ ] Бот игнорирует повторные сигналы на той же свече (защита через `processed_candles`).

## 3. Риск-менеджмент (DCA и Stop-Loss)
- [ ] Выставляется защитный стоп-лосс -25%.
- [ ] Выполняется до 3-х усреднений (DCA) при падении цены.
- [ ] Частичное закрытие позиции (Take Profit 1, 2, 3) срабатывает корректно.

## 4. База данных и логи
- [ ] Данные о сделках сохраняются в `tradesv3.sqlite`.
- [ ] Файл состояния `strategy_state.json` обновляется после каждой свечи.
